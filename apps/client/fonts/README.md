# Bundled fonts (CL-0b)

The brand typefaces are bundled as variable `.ttf` in
`src/commonMain/composeResources/font/` and applied via `theme/Theme.kt`
(`brandTypography()`):

- **Outfit** — display / headline / title (`outfit.ttf`)
- **Figtree** — body / label (`figtree.ttf`)

Both are from [Google Fonts](https://fonts.google.com) under the **SIL Open Font
License 1.1** (see `OFL.txt`). The OFL permits bundling/redistribution in software;
the fonts are not sold separately and retain their names per the license.
